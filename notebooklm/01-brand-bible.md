## Design System: NeoBrutalism

DMC Kreatif uses a bold **NeoBrutalist** design language — this is critical for all visual content.

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| **Neo Lime** (PRIMARY) | #CDFF50 | Primary accent, CTAs, highlights, logo background |
| **Neo Black** | #121212 | Text, borders, shadows |
| **Neo Background** | #FFFDF5 | Cream/off-white page background |
| **Neo Background Alt** | #F5F0E8 | Section alternating background |
| **Neo White** | #FFFFFF | Cards, clean surfaces |
| **Neo Yellow** | #FBFF48 | Secondary accent, e-commerce service |
| **Neo Pink** | #FF70A6 | Tertiary accent, marketing service |
| **Neo Blue** | #3B82F6 | SEO service, trust elements |
| **Neo Green** | #33FF57 | Status indicators, success |
| **Neo Purple** | #A855F7 | Premium/Commerce tier |
| **Neo Orange** | #FF9F1C | Warnings, attention |
| **Neo Red** | #FF2A2A | Errors, urgency |

### Typography
- **Headings:** Space Grotesk (Bold, 700 weight) — ALL UPPERCASE with tight tracking
- **Body/Monospace:** JetBrains Mono — used for descriptions, labels, technical text
- **Hero text:** Fluid sizing from 3rem to 8rem (clamp)
- **All headings:** Uppercase, bold, letter-spacing tight

### Visual Elements
- **Borders:** 2-4px solid black (#121212) on EVERYTHING
- **Shadows:** Hard offset shadows (no blur) — `4px 4px 0px 0px #121212`
- **Shadow sizes:** sm (2px), default (4px), lg (8px), xl (12px)
- **Background pattern:** Dot grid pattern — `radial-gradient(#121212 1px, transparent 1px)` at 40px spacing
- **Cards:** White background, thick black border, hard shadow
- **Buttons:** Filled with thick border, hard shadow, press effect (shadow reduces on hover/click)
- **No rounded corners** — everything is sharp/square
- **No gradients** — flat solid colors only
- **Decorative shapes:** Scattered colored rectangles and squares at page edges

### Animation Style
- Fade-in-up on scroll
- Staggered container animations
- Marquee ticker text
- Glitch effect on hover
- Press effect on buttons (translate + shadow reduction)
- Respects prefers-reduced-motion

### Logo Description (for AI prompts)
The DMC Kreatif logo is a vertically stacked dual-banner design:
1. **Top banner:** Lime green (#CDFF50) rectangle with 3px black border, containing "DMC KREATIF" in bold black uppercase Space Grotesk font with 2px letter spacing
2. **Bottom banner:** White (#FFFFFF) rectangle with 3px black border (connected to top), containing "GMG DESIGN" in bold black uppercase Space Grotesk font with 2px letter spacing
3. **Shadow:** Both banners cast a hard black drop shadow (offset, no blur)
4. **Size:** Rectangular, wider than tall (approximately 280x80 ratio)

## Brand Voice & Tone

### Personality
- **Confident** — "We build websites that convert" not "We try to make good websites"
- **Technical but accessible** — Uses code/dev terminology as style (SYS.SERVICES, >_ SERVICE_01) but remains understandable
- **Direct** — Short, punchy sentences. No fluff.
- **Premium** — Positions as high-quality, not cheap. Uses words like "handcrafted", "premium", "professional"
- **European** — Emphasizes serving European businesses, multilingual capability

### Language Style
- Headlines: UPPERCASE, SHORT, PUNCHY (max 5-6 words)
- Subheadings: Monospace style with system prefixes (SYS.SERVICES, SYS.PRICING)
- CTAs: Action-oriented imperatives ("START YOUR PROJECT", "GET A FREE QUOTE", "VIEW OUR WORK")
