## Slide Layout Rules

### Background
- Primary: Cream (#FFFDF5) with subtle dot grid pattern
- Alternative: Solid black (#121212) for contrast/urgency slides
- NEVER use gradients, blurs, or photographic backgrounds
- NEVER use rounded corners anywhere

### Borders & Shadows
- EVERY card, box, button, banner MUST have a thick black border (3-4px)
- EVERY elevated element MUST have a hard drop shadow: `offset 4-8px right, 4-8px down, solid black, NO BLUR`
- Shadow gives a "stacked paper" or "sticker" effect

### Typography Rules
- ALL headings: UPPERCASE, bold, Space Grotesk or similar geometric sans-serif
- Body text: Monospace font (JetBrains Mono style) for technical/label feel
- Maximum 3-4 words per headline for impact
- Code-style labels like `>_ SERVICE_01` or `SYS.PRICING` for section markers
- Letter spacing: tight for headings, wide for labels

## WhatsApp Status Specific Rules (1080x1920 Vertical)

### Structure (top to bottom):
1. **Top 15%:** Logo + status badge
2. **25%:** Main headline (2-3 words, HUGE font)
3. **35%:** Content area (services grid / pricing / stats)
4. **15%:** CTA button
5. **10%:** Website URL + micro-copy

### Visual Impact Requirements:
- Must be readable at phone screen size (not tiny text)
- Minimum font size for body: 28px equivalent
- Headlines: 52px+ equivalent
- High contrast: black on lime, or lime on black
- One clear message per status ‚Äî don't overcrowd

## Decorative Elements to Include

- Small colored squares (20-80px) scattered near corners
- Occasionally rotated 10-15 degrees
- Colors: random from palette (lime, yellow, pink, blue)
- NEVER more than 4-5 decorative shapes per slide
- They should feel like "stickers" ‚Äî with black borders and hard shadows
- Status indicator: small green square + monospace text "AVAILABLE FOR PROJECTS"
